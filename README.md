 Store 316 - Exclusive E-commerce
Este é um projeto de e-commerce de alto padrão focado em experiência do usuário e performance. O sistema oferece uma interface de boutique minimalista e uma área administrativa robusta para gestão de produtos.

// Tecnologias
Front-end: Next.js 16 (App Router), TypeScript, Tailwind CSS.

Backend & Banco de Dados: Supabase (PostgreSQL).

Autenticação: Iron Session.

Ícones: Lucide React.

// Principais Funcionalidades
Vitrine Dinâmica: Carregamento de produtos por categoria com design de luxo.

Painel Administrativo: Gestão completa de inventário (CRUD), incluindo upload de múltiplas imagens via Google Drive.

Sacola Inteligente: Controle global de estado para adicionar itens com seleção de tamanhos específicos.

Checkout Customizado: Geração automática de pedidos formatados para WhatsApp.

Segurança: Rotas administrativas protegidas por Middleware e validação de sessão no servidor.

// Como executar o projeto
Clone o repositório.

Instale as dependências com npm install.

Configure o arquivo .env com suas chaves do Supabase.

Execute npm run dev.
